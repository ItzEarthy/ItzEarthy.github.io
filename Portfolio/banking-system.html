Banking System (C++)

This is a simple banking system implemented in C++. It enables users to manage bank accounts through a console-based interface, with persistent data stored via file I/O.

🔧 Features

  • Account creation with unique IDs & passwords
  • Secure login system
  • Deposit and withdraw funds
  • Transfer money between users
  • Account deletion

💻 Usage

  1. Clone the repository: git clone https://github.com/ItzEarthy/BankingSystemProject
  2. Compile the files using a C++ compiler: 
    g++ main.cpp bankfunctions.cpp -o BankApp
  3. Run the executable: ./BankApp
